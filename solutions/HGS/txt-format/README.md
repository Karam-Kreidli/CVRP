# Solutions/HGS/txt-Format Folder

## Why This Folder Exists

This folder stores `.txt` duplicates of baseline HGS route files.

## File Format

Content matches `.sol` outputs but uses `.txt` extension for tool compatibility and easy preview.

- filename pattern: `<instance>.txt`
- one route per line (`Route #k: ...`)

## Typical Producer

- `Baseline/Portfolio_Solver.py`
