# Solutions/HGS Folder

## Why This Folder Exists

`Solutions/HGS/` stores route exports generated by pure HGS baseline workflows.

It exists to provide a non-RL reference output set for quality checks and comparisons.

## What This Folder Contains

```text
Solutions/HGS/
  sol-Format/
  txt-Format/
```

## Typical Producer

Primary script:

- `Baseline/Portfolio_Solver.py`

Default output command:

```bash
python Baseline/Portfolio_Solver.py \
  --instance_path Data \
  --output_dir Solutions/HGS
```

## Output Semantics

- one file per instance in `.sol` and `.txt` forms,
- same route content in both formats,
- route numbering starts at 1.

## Why Keep This Separate from `HGS+RL`

Separation supports clean A/B analysis:

- `HGS/` = baseline-only behavior,
- `HGS+RL/` = RL-guided behavior.
