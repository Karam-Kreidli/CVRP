# Solutions/HGS+RL/txt-Format Folder

## Why This Folder Exists

This folder stores `.txt` duplicates of RL-guided route files.

## File Format

Content is equivalent to `.sol` exports, with `.txt` extension for portability and quick inspection.

- filename pattern: `<instance>.txt`
- one route per line (`Route #k: ...`)

## Typical Producer

- `Baseline/Infer.py` (when solution export is enabled)
