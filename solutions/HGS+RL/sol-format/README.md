# Solutions/HGS+RL/sol-Format Folder

## Why This Folder Exists

This folder stores `.sol` route files generated by RL-guided inference runs.

## File Format

Each file is plain text, one route per line:

```text
Route #1: 35 46 31
Route #2: 15 22 41 20
```

- filename pattern: `<instance>.sol`
- reflects best route set found by environment tracking

## Typical Producer

- `Baseline/Infer.py` (when solution export is enabled)
