# Solutions/HGS+RL Folder

## Why This Folder Exists

`Solutions/HGS+RL/` stores route exports generated by the RL-guided inference pipeline.

It exists to preserve route-level artifacts from policy-guided runs separately from pure baseline outputs.

## What This Folder Contains

```text
Solutions/HGS+RL/
  sol-Format/
  txt-Format/
```

## Typical Producer

Primary script:

- `Baseline/Infer.py`

Example command:

```bash
python Baseline/Infer.py \
  --instance_dir Data \
  --model_path Logs/Best_Model.pth \
  --solution_output_dir Solutions/HGS+RL
```

If `--baseline` is also enabled, this folder still stores RL routes while comparison reports are written to `Results/`.

## Output Semantics

- one file per instance in `.sol` and `.txt` forms,
- route lines reflect best routes tracked across the environment episode,
- folder structure mirrors `Solutions/HGS/` for direct comparison.

## Why Keep This Separate from `HGS`

- clear provenance of RL-guided artifacts,
- simpler diff/inspection workflows,
- avoids accidental mixing when benchmarking.
