## Notes

- **ILS-SP**: Iterated Local Search with Set Partitioning (Subramanian et al., 2012)
- **UHGS**: Unified Hybrid Genetic Search (Vidal et al., 2012) — the algorithm used by `hygese`
- **BKS**: Best Known Solution from BCP (Branch-Cut-and-Price) or best of all methods
- **NV** column shows BKS fleet size; note some instances have NV > k (e.g., X-n148-k46 has NV=47)
- Source: Uchoa et al., "New benchmark instances for the Capacitated Vehicle Routing Problem", 2014
